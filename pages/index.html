You have probably heard that the closer you are to the equator, the hotter the temperature is. Does this hold true? If so, does the proximity to the equator also affect other weather-related variables. On this website you can examine how the proximity to the equator affects:

  • (Max) Temperature
  • Humidity
  • Cloudiness
  • Wind Speed

We pulled data on over 500 cities from the OpenWeatherMap API. The full list of cities and all relevant data can be found here. After retrieving the data, we used the python library pandas to organize the data into a dataframe. Thereafter we used the python library Matplotlib to plot selected columns from our dataframe, and thereby visually analyze whether there are any relationships between the weather-related variables.
